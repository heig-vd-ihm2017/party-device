# Party device
A device for parties with lights that react to sounds.

## Cloning the repository and initialize modules
```
git clone git@github.com:heig-vd-ihm2017/party-device.git
cd party-device
git submodule init
git submodule update
```

### Updating modules
```
cd code/lib/<module>
git checkout master
git pull
```
